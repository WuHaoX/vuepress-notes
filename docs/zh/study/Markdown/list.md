# 列表语法
可以将多个条目组织成有序或无序列表
## 有序列表
要创建有序列表，需要在每个列表项前添加数字并紧跟一个英文句点。每列数字不必按大小顺序排列，但时列表应该从数字1开始。

| 1. 第一列<br>1. 第二列<br>1. 第三列<br>1. 第四列                                      | &lt;ol&gt;<br>&lt;li&gt;第一列&lt;/li&gt;<br>&lt;li&gt;第二列&lt;/li&gt;<br>&lt;li&gt;第三列&lt;/li&gt;<br>&lt;li&gt;第四列&lt;/li&gt;<br>&lt;/ol&gt;                                                                                                        | <ol><li>第一列</li><li>第二列</li><li>第三列</li><li>第四列</li></ol>                                                                    |

## 有序列表需要注意的地方
在CommonMark或者一些其它Markdown应用程序里会使用（ ) ）符号代替英语巨点（ . ）但并不是所有的Markdown应用程序都支持这种方法，所有最好还是用英文句点（ . ）
| :heavy_check_mark:正确的做法 |    :x:错误的做法    |
|:----------------------------:|:-------------------:|
| 1. 列表一 <br>2. 列表二          | 1) 列表一<br> 2) 列表二 |
## 无序列表
要创建无序列表，请在每个列表项前面添加破折号（ - ）、星号（ * ）或加号（ + ）。缩进一个或多个列表可创建嵌套列表。
|                                  Markdown语法                                  |                                                                                                                             HTML                                                                                                                             |                                                                           预览效果                                                                           |

| - 第一列<br>- 第二列<br>- 第三列<br>- 第四列                                   | &lt;ol&gt;<br>&lt;li&gt;第一列&lt;/li&gt;<br>&lt;li&gt;第二列&lt;/li&gt;<br>&lt;li&gt;第三列&lt;/li&gt;<br>&lt;li&gt;第四列&lt;/li&gt;<br>&lt;/ol&gt;                                                                                                        | <ul><li>第一列</li><li>第二列</li><li>第三列</li><li>第四列</li></ul>                                                                    |
| * 第一列<br>* 第二列<br>* 第三列<br>* 第四列                                   | &lt;ul&gt;<br>&lt;li&gt;第一列&lt;/li&gt;<br>&lt;li&gt;第二列&lt;/li&gt;<br>&lt;li&gt;第三列&lt;/li&gt;<br>&lt;li&gt;第四列&lt;/li&gt;<br>&lt;/ul&gt;                                                                                                        | <ul><li>第一列</li><li>第二列</li><li>第三列</li><li>第四列</li></ul>                                                                    |

## 无序列表需要注意的地方
Markdown应用程序不支持同一列表中有不同的符号，所以在一个列表中最好只有一种符号。
|         :heavy_check_mark:正确的做法         |                 :x:错误的做法                |
|:--------------------------------------------:|:--------------------------------------------:|
| - 第一列<br>- 第二列<br>- 第三列<br>- 第四列 | - 第一列<br>* 第二列<br>+ 第三列<br>- 第四列 |
## 在列表嵌套其它元素
要在保留列表连续性的同时在列表中添加另一种元素，请将该元素缩进四个空格或一个制表符。 
如下列所示：  
### 段落
```
1. 第一列
2. 第二列
    ***插入的其它元素***
3. 第三列
```
渲染效果如下
1. 第一列
2. 第二列  
    ***插入的其它元素***
3. 第三列

PS:**在添加处的上一行列表处注意使用**[换行语法](./lineFeed.md) 
### 引用块
```
1. 第一列
2. 第二列
    > 插入的其它元素
3. 第三列
```
渲染效果如下
1. 第一列
2. 第二列
    > 插入的其它元素
3. 第三列
### 代码块
[代码块](./code.md)通常采用四个空格或一个制表符缩进。当他们被放在列表中时，请将它们缩进八个空格或两个制表符。
```
1. 第一列
2. 第二列
        `
        <html>
            <head>
                <title>Test</title>
            </head>
        </html>
        `
3. 第三列
```
渲染效果如下
1. 第一列
2. 第二列  
        `
        <html>
            <head>
                <title>Test</title>
            </head>
        </html>
        `
3. 第三列
### 图片
```
1. 第一列
2. 第二列
![hand](../../../../docs/.vuepress/public/images/logo.png)
3. 第三列
```
渲染效果如下
1. 第一列
2. 第二列
![图片加载中](../../../../docs/.vuepress/public/images/hero.png)  
3. 第三列

PS:需要注意的是图片在Markdown里不能设置宽高，需要用HTML`<img />`标签来设置图片的宽高
### 列表
```
1. 第一列表
2. 第二列表
    - 嵌套第一列表
    - 嵌套第二列表
3. 第三列表
```
渲染效果如下
1. 第一列表
2. 第二列表
    - 嵌套第一列表
    - 嵌套第二列表
3. 第三列表
